## 什么是单例模式
单例模式（Singleton pattern）：确保一个类只有一个实例，并提供该实例的全局访问点。属于创建型模式。

## 功能简介
解决一个全局使用的类频繁地创建与销毁，控制实例数目，节省系统资源。
